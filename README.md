# react-rails redux sample
this is a simple example application using  react-rails and redux
* rails 4.2.3
* redux 3.0.4
* react-rails 1.4.2
* react 0.14.2

## Installation

```
git clone https://github.com/suzan2go/react-rails-redux-sample.git
cd react-rails-redux-sample
bundle install
npm install
```
